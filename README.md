# ✨ AI 编程提示词生成器

一个Chrome浏览器扩展，能够从网页元素生成精确的AI编程指令。通过可视化的元素选择，将网页上的任意组件转换为高质量、详细的AI提示词，让AI助手能够生成几乎一模一样的代码。

## 🎯 核心功能

### 🔍 智能元素选择
- 鼠标悬停高亮显示页面元素
- 点击选择目标组件
- 支持复杂嵌套结构的精确提取

### 🧠 智能提示词生成
- 自动提取HTML结构和CSS样式
- 生成详细的自然语言描述
- 包含完整的技术要求和实现指南

### 📋 一键复制分享
- 一键复制生成的提示词
- 直接粘贴给AI助手使用
- 支持ChatGPT、Claude等主流AI工具

## 🚀 快速开始

### 安装方法

1. **下载项目**
   ```bash
   git clone [repository-url]
   cd Chrome_PRD_For_AI
   ```

2. **加载到Chrome**
   - 打开Chrome浏览器
   - 访问 `chrome://extensions/`
   - 开启"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择项目中的 `public` 文件夹

3. **开始使用**
   - 在任意网页点击插件图标
   - 选择"打开 AI 编程提示词生成器"
   - 侧边栏将会打开

### 使用方法

1. **激活选择模式**
   - 点击"🎯 选择元素"按钮
   - 页面右上角会显示选择模式提示

2. **选择目标元素**
   - 在网页上移动鼠标，元素会高亮显示
   - 点击您想要复制的元素

3. **获取AI指令**
   - 插件自动生成详细的编程指令
   - 点击"📋 复制提示词"复制到剪贴板

4. **使用指令**
   - 将指令粘贴给任何AI助手
   - AI将根据指令生成相应的代码

- 详细的技术要求说明
- 包含参考信息和使用建议

### 现代化的用户界面
- 清晰直观的操作流程
- 实时状态反馈
- 响应式设计，适配不同屏幕

## 📖 使用示例

### 示例1：复制GitHub按钮

**操作步骤：**
1. 访问 GitHub.com
2. 激活选择模式
3. 点击"Sign up"按钮
4. 复制生成的提示词

**生成的指令示例：**
```
你好，请使用HTML和CSS为我创建一个组件。请严格按照以下描述的结构和样式来编写代码，不要使用任何未经说明的样式。

## 1. HTML 结构

请创建一个 a 标签，class 为 "btn-mktg"。
这个元素包含文本内容："Sign up for GitHub"

## 2. CSS 样式

针对 a.btn-mktg 元素的样式：

**布局属性：**
  - display: inline-block

**尺寸属性：**
  - padding: 12px 20px

**颜色和背景：**
  - background-color: #238636
  - color: #ffffff

**字体样式：**
  - font-weight: 600
  - font-size: 16px

**边框和效果：**
  - border-radius: 6px
  - box-shadow: 0 1px 0 rgba(27, 31, 36, 0.1)

## 3. 技术要求

- 请提供完整的、可直接运行的HTML和CSS代码
- 确保代码语法正确且符合现代Web标准
- 保持响应式设计原则
- 不要添加任何JavaScript代码，除非明确要求
- 请使用语义化的HTML标签

请开始创建代码。
```

## 🛠️ 技术架构

### 核心组件

- **Content Script (`content.js`)**：负责页面元素的选择和数据提取
- **Background Script (`background.js`)**：处理插件间通信和状态管理
- **Side Panel (`sidepanel.js`)**：用户界面和提示词生成逻辑
- **Styling (`sidepanel.css`)**：现代化的UI样式

### 关键技术

- **Element Selection**：基于DOM事件的精确元素定位
- **Style Extraction**：使用`getComputedStyle`获取最终样式
- **Intelligent Parsing**：智能分析和分类CSS属性
- **Natural Language Generation**：将技术信息转换为自然语言

## 🌟 最佳实践

### 选择元素时
- 优先选择设计精美、结构清晰的组件
- 避免过度复杂的嵌套结构（>10层）
- 选择有明确视觉边界的元素

### 使用生成的指令时
- 复制完整的指令给AI助手
- 可以根据需要调整文本内容
- 建议先测试简单元素，再尝试复杂组件

### 提升效果的技巧
- 选择主流网站的成熟组件
- 关注响应式设计良好的元素
- 验证生成的代码在不同浏览器中的效果
